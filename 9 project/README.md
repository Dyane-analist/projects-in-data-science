# Проект "Золотодобыча": 
Девятый учебный проект, направленный на отработку навыков  анализа данных и машинного обучения: анализ и предобработка данных, работа с выбросами, подготовка признаков, обучение моделей, подбор оптимальных гиперпараметров и выбор лучшей модели на основании собственной качественной метрики, тестирование модели и сравнение результата с константной моделью.
