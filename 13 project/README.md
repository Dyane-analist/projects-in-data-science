# Проект "Описания товаров": 
Тринадцатый учебный проект, направленный на приобретение навыков машинного обучения для работы с текстами (оценка тональности комментария): анализ и предобработка данных (очистка текстов корпуса, векторизация текстов с помощью модели DistilBERT с встроенным токенизатором), обучение моделей с учетом порога классификации, подбор оптимальных гиперпараметров и выбор лучшей модели на основании качественной метрики F1-score, проверка модели на валидационной и тестовой выборках.
